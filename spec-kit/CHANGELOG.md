# Changelog

All notable changes to the Spec2Cloud spec-kit (extension, preset, workflow) are documented in this file. Each entry corresponds to a published GitHub Release with the tag `spec-kit-spec2cloud-v<version>`. The release version is the **highest semver** among `extension.yml`, `preset.yml`, and `workflow.yml`.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/), and this project follows [Semantic Versioning](https://semver.org/) per artifact.
